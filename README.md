# RAG-Powered Complaint Analysis Chatbot

This project builds an intelligent complaint-analysis chatbot for financial services using
Retrieval-Augmented Generation (RAG).

The system analyzes customer complaints across multiple financial products and allows
internal teams to ask natural-language questions and receive evidence-backed insights.
